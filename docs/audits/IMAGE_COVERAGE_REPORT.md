## Resolution & Authenticity
As per the strict Data Authenticity constraints, **no images were generated, hallucinated, or faked**. If a verified portrait does not exist in the public API, the application guarantees that a broken image link is never rendered. Instead, it utilizes a deeply thematic, programmatic "Premium Fallback Portrait" that renders the character's initials above their House Sigil, wrapped in a unique CSS gradient matching their house colors.
